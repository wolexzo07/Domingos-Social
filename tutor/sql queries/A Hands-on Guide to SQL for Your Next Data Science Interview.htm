A Hands-on Guide to SQL for Your Next Data Science Interview

Because data science starts with the database

Abhinav Sagar
Follow
Oct 29 · 4 min read

SQL is a database query language designed for the retrieval and management of data in a relational database. It is the standard language for relational database management systems. A relational database management system (RDBMS) is the basis for SQL, and for all modern database systems such as MS SQL Server, Oracle and MySQL. The relational database system contains one or more objects called tables and data is stored in these tables in the form of rows and columns.

Some of the advantages of using SQL are:

  1. Portable: SQL runs on mainframes, PCs, laptops, servers, tablets and smartphones.
  2. Open Source: SQL databases which are free and open source can be used at low cost and with large communities behind them.
  3. SQL a highly desirable skill: Many jobs including data science, IT support, web development and business data analysis require skills in SQL.
  4. Easy to learn and understand: SQL consists mainly of English statements which makes it easy to learn and write SQL queries.
  5. Client/Server language: SQL is used to connect front end computers (clients) with back end databases (servers) thus supporting the client-server architecture.

Tables

Let’s say we have 2 tables — StudentDetails containing the details of the students enrolled in a college and StudentMarks containing marks of the students in various subjects registered by them. These tables as shown below are taken as reference throughout this article.

SQL Queries

1. Write a SQL query to fetch the count of students studying Maths.

SELECT COUNT(*) FROM StudentMarks WHERE Subject = 'Maths';

2. Write A SQL Query To Fetch “Full_Name” From “StudentDetails” Table In Upper Case.

SELECT UPPER(Full_Name) FROM StudentDetails;

3. Write A SQL Query To Fetch Unique Values Of “Subject” From “StudentMarks” Table.

SELECT DISTINCT Subject FROM StudentMarks;

4. Write A SQL Query To Print The “Full_Name” From “StudentDetails” After Replacing ‘a’ With ‘A’.

SELECT REPLACE(Full_Name,'A','a') FROM StudentDetails;

5. Write a query to fetch only the first name from the “Full_Name” column of “StudentDetails” table.

SELECT MID(Full_Name, 0, LOCATE(' ',Full_Name)) FROM StudentDetails;

6. Write A SQL Query To Print All Student Details From The “StudentDetails” Table Order By “Full_Name” in ascending order.

SELECT * FROM StudentDetails ORDER BY Full_Name ASC;

7. Write A SQL Query To Print All Student Details With The “Full_Name” As “Sneha Singh” And “Nikhil Mishra” From “StudentDetails” Table.

SELECT * FROM StudentDetails WHERE Full_Name IN ('Sneha Singh','Nikhil Mishra');

8. Write a SQL query to fetch subject wise count of students sorted by subject’s count in descending order.

SELECT Subject, count(Student_Id) StudentSubjectCount FROM StudentMarks GROUP BY Subject ORDER BY StudentSubjectCount DESC;

9. Write A SQL Query To Print Details Of The Students Whose “Full_Name” Contains ‘a’.

SELECT * FROM StudentDetails WHERE Full_Name LIKE '%a%';

10. Write A SQL Query To Fetch The Count Of Students Studying Physics.

SELECT COUNT(*) FROM StudentMarks WHERE Subject = 'Physics';

11. Write A SQL Query To Print Details Of The Students Whose “Date_of_Joining” Lies Between “20/10/2019” And “25/10/2019”.

SELECT * FROM StudentDetails WHERE Date_of_Joining BETWEEN '20/10/2019' AND '25/10/2019';

12. Write a SQL query to fetch duplicate records from “StudentDetails” table.

SELECT * FROM StudentDetails HAVING COUNT(*) > 1;

13. Write a SQL query to create a new table with data and structure copied from another table.

SELECT * INTO NewTable FROM StudentDetails;

14. Write An SQL Query To Fetch The No. Of Students Studying Each Subject In The Descending Order.

SELECT Subject, COUNT(*) No_Of_Students FROM StudentMarks
GROUP BY Subject ORDER BY No_Of_Students DESC;

15, Write An SQL Query To Show Only Even Rows From “StudentDetails” Table.

SELECT * FROM StudentDetails WHERE MOD (Student_ID, 2) = 0;

16. Write A SQL Query To Show The Top 3 Records Of “StudentMarks” Table.

SELECT * FROM StudentMarks ORDER BY Marks DESC LIMIT 3;

17. Write SQL query to find the 3rd highest marks from “StudentMarks” table.

SELECT * FROM StudentMarks ORDER BY Marks DESC LIMIT 2,1;

18. Write A SQL Query To Fetch The List Of Students With The Same Marks.

SELECT DISTINCT S.Student_ID, S.Subject, S.Marks 
FROM StudentMarks S, StudentMarks S1 WHERE S.Marks = S1.Marks 
AND S.Student_ID != S1.Student_ID;

19. Write A SQL Query To Show All Subjects Along With The Number Of Students In There.

SELECT Subject, COUNT(Subject) AS 'Number of Students' FROM StudentMarks GROUP BY Subject;

20. Write A SQL Query To Fetch The Names Of Students Who Scored The Highest Marks.

SELECT * FROM StudentMarks WHERE Marks=(SELECT Max(Marks) FROM StudentMarks);
